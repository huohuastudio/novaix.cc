# 隐私政策

生效日期：2026 年 6 月 22 日

Spark Studio（以下简称"我们"）重视您的隐私。本隐私政策说明了我们在您使用 Novaix 软件及相关服务（包括官网、文档站、授权中心，以下统称"服务"）时如何收集、使用、存储和保护您的信息。

我们依据《中华人民共和国个人信息保护法》《中华人民共和国数据安全法》《中华人民共和国网络安全法》及相关法律法规制定本政策。

## 1. 我们收集的信息

### 1.1 通过授权中心收集的信息

当您注册授权中心账户或购买许可证时，我们会收集：

- **账户信息**：邮箱地址、密码（以不可逆哈希形式存储）
- **交易信息**：订单编号、购买的许可证类型、支付金额、支付时间。具体的支付账户信息（如银行卡号）由第三方支付服务商处理，我们不存储此类信息

### 1.2 通过软件许可验证收集的信息

Novaix 软件在运行过程中会定期与授权中心通信以验证许可证有效性。此过程中传输的数据仅包括：

- 许可证密钥
- 授权域名（您配置的外部访问地址所对应的域名）
- 软件版本号

### 1.3 通过网站自动收集的信息

当您访问我们的网站（novaix.cc）时，我们通过 Google Analytics 收集匿名的访问统计数据，包括页面浏览量、访问时长、大致地理位置、设备类型和浏览器类型。这些数据不包含个人身份信息。

Google Analytics 的数据处理受 [Google 隐私政策](https://policies.google.com/privacy)约束。由于 Google Analytics 的服务器位于境外，相关匿名统计数据会传输至境外处理。

### 1.4 通过反馈渠道收集的信息

当您通过邮件联系我们或在 GitHub 提交 Issue 时，我们会收集您提供的沟通内容和联系方式。GitHub Issue 的处理受 [GitHub 隐私声明](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement)约束。

### 1.5 我们不收集的信息

我们明确声明，以下信息**不会被收集、传输或上报**：

- 您使用 Novaix 软件管理的最终用户信息
- 您服务器上的实例、订单、财务等业务数据
- 您服务器的系统配置、性能指标或日志
- 任何形式的用户行为追踪或使用习惯数据

Novaix 软件本身不包含遥测、数据上报或行为追踪功能。软件的更新检查通过 GitHub 公开 API 进行，不涉及许可证数据或个人信息的传输。

## 2. 我们如何使用信息

我们收集的信息用于以下目的：

| 用途 | 法律依据 |
|------|---------|
| 处理许可证验证和发放 | 履行合同义务 |
| 提供软件更新和技术支持 | 履行合同义务 |
| 处理购买交易和发送交易通知 | 履行合同义务 |
| 发送许可证到期提醒和安全更新通知 | 履行合同义务 |
| 通过匿名统计数据改进网站体验 | 合法利益 |
| 遵守适用法律法规的要求 | 法律义务 |

我们不会将您的个人信息用于自动化决策或用户画像。

## 3. 信息存储与安全

3.1 您的账户信息和交易数据存储在位于中国境内的服务器上。

3.2 我们采取以下措施保护您的信息：

- 所有数据传输使用 HTTPS/TLS 加密
- 密码使用不可逆的哈希算法存储
- 许可证验证使用 Ed25519 数字签名确保证书完整性
- 服务器访问权限严格控制，遵循最小权限原则

3.3 尽管我们尽力保护您的信息，但没有任何传输或存储方法是绝对安全的。如果发生数据安全事件，我们将按照法律要求及时通知受影响的用户和相关监管部门。

## 4. 数据保留

- **账户信息**：在您的账户存续期间保留。账户注销后，我们将在三十（30）天内删除您的个人信息，但法律要求保留的除外
- **交易记录**：根据《中华人民共和国会计法》等法规要求，财务相关记录至少保留五（5）年
- **许可验证数据**：仅用于实时验证，不做持久化存储
- **网站访问统计**：Google Analytics 按其数据保留政策处理，默认保留期限为十四（14）个月

## 5. 信息共享与披露

5.1 我们**不会**出售、出租或交易您的个人信息。

5.2 在以下情况下，我们可能会共享您的信息：

- **支付处理**：与您选择的第三方支付服务提供商共享必要的交易信息以完成支付
- **网站分析**：通过 Google Analytics 收集的匿名统计数据（不含个人身份信息）
- **法律要求**：当法律法规、法律程序或有权政府机关要求时
- **权利保护**：为保护我们、用户或公众的权利、财产或安全所合理必需时

5.3 除网站分析数据外，我们目前不主动向境外传输个人信息。如未来涉及跨境数据传输，我们将按照《个人信息保护法》的要求采取必要的保护措施并事先通知您。

## 6. Cookie 与追踪技术

6.1 我们的网站使用 Google Analytics 提供的 Cookie 来收集匿名访问统计信息。

6.2 您可以通过浏览器设置管理或禁用 Cookie。禁用 Cookie 不会影响您正常使用我们的网站和文档。

6.3 本软件本身不使用 Cookie 或任何追踪技术。

## 7. 您的权利

根据《个人信息保护法》及相关法律，您享有以下权利：

- **知情权和决定权**：了解我们对您个人信息的处理情况，并有权决定是否同意
- **查阅和复制权**：查阅和复制我们持有的您的个人信息
- **更正权**：要求更正不准确或不完整的个人信息
- **删除权**：在法定情形下要求删除您的个人信息
- **账户注销权**：注销您在授权中心的账户
- **撤回同意权**：撤回您此前给予的同意（不影响撤回前已进行的处理）

如需行使上述权利，请通过本政策末尾的联系方式与我们联系。我们将在十五（15）个工作日内处理您的请求。

## 8. 未成年人保护

我们的服务面向具有完全民事行为能力的主体。我们不会故意收集不满十四周岁未成年人的个人信息。如果我们发现无意中收集了此类信息，将立即采取措施删除。

## 9. 政策更新

9.1 我们可能会根据业务发展和法律要求更新本隐私政策。更新后的政策将在我们的网站上公布，并注明新的生效日期。

9.2 对于涉及您权利的重大变更，我们会通过网站公告或邮件方式提前通知您。
